Desarrollo de Aplicaciones Multiplataforma

Introduccion al ciclo

El ciclo DAM (Desarrollo de Aplicaciones Multiplataforma) está diseñado para formar profesionales
en la creación de aplicaciones para diferentes plataformas, incluyendo dispositivos móviles y ordenadores.

Plan de Estudios

1° Año

La asignatura de Programación en DAM enseña los fundamentos del desarrollo de software. Aprenderás a escribir código en lenguajes como Java, Python o C#, utilizando estructuras de control, funciones, programación orientada a objetos (POO) y manejo de bases de datos. También desarrollarás lógica de programación y resolverás problemas mediante algoritmos eficientes.

La asignatura de Base de Datos en DAM se centra en el diseño, gestión y administración de datos. Aprenderás a crear y manejar bases de datos relacionales con SQL, utilizando sistemas como MySQL o PostgreSQL. También estudiarás normalización, relaciones entre tablas y el uso de CRUD (Create, Read, Update, Delete) para gestionar información de forma eficiente.

La asignatura de Lenguajes de Marcas en DAM trata sobre la estructuración y presentación de la información en la web y otros entornos. Aprenderás a utilizar HTML y XML para organizar datos, junto con CSS para el diseño y estilo. También verás el uso de JSON y YAML para el intercambio de información.

2° Año

La asignatura de Desarrollo de Interfaces en DAM se enfoca en la creación de interfaces gráficas para aplicaciones. Aprenderás a diseñar y programar UI/UX con herramientas como JavaFX, Swing o Android Studio, utilizando patrones de diseño y adaptando interfaces a diferentes dispositivos.

La asignatura de Programación de Servicios y Procesos en DAM se centra en el desarrollo de aplicaciones que trabajan con múltiples procesos y servicios. Aprenderás a gestionar hilos (multithreading), concurrencia y sincronización, así como a crear y consumir servicios web (REST y SOAP). También se estudia la comunicación entre aplicaciones y la ejecución de tareas en segundo plano.

La asignatura de Acceso a Datos en DAM se enfoca en la gestión y manipulación de bases de datos desde aplicaciones. Aprenderás a conectar programas con bases de datos SQL (MySQL, PostgreSQL) y NoSQL (MongoDB), utilizando tecnologías como JDBC, Hibernate o JPA. También trabajarás con XML y JSON para el intercambio de información.

Salidas Profesionales y Empresas con salidas

Salidas Profesionales

Focalizado en la creación y mantenimiento de páginas y aplicaciones web, utilizando tecnologías como HTML, CSS, JavaScript, frameworks como React o Angular, y lenguajes de programación backend como PHP, Python o Ruby.

Especializado en la creación de aplicaciones para dispositivos móviles, ya sea para iOS, Android o plataformas híbridas.

Encargado de escribir, probar y mantener el código de aplicaciones y programas de software, tanto para empresas como para proyectos propios.

Empresas

Puedes trabajar como desarrollador de software o aplicaciones, creando soluciones para clientes en distintos sectores.

Trabajarías en el desarrollo de aplicaciones a medida, implementando tecnologías móviles y multiplataforma para diferentes sectores.

Desarrollador de aplicaciones móviles y web, trabajando en proyectos de transformación digital para empresas y administraciones públicas.
